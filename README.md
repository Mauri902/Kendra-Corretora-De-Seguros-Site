# 🛡️ Kendra Corretora de Seguros

> Sistema web desenvolvido para gerenciamento de clientes e apoio ao atendimento de uma corretora de seguros.

---

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de criar um sistema web para a **Kendra Corretora de Seguros**, permitindo o cadastro, organização e gerenciamento de dados de clientes de forma simples e eficiente.

A aplicação funciona como uma **ferramenta administrativa**, onde a corretora registra informações dos clientes para facilitar:

- 📋 Organização de dados
- 📞 Atendimento personalizado
- 📊 Controle de informações
- 🧾 Suporte a cotações de seguros

---

## 🎯 Objetivo

O principal objetivo do sistema é:

> Auxiliar a corretora no gerenciamento de clientes, centralizando informações importantes em um único sistema, tornando o atendimento mais ágil, organizado e profissional.

---

## 🏢 Sobre a Corretora

A **Kendra Corretora de Seguros** atua oferecendo atendimento personalizado e consultoria em seguros, buscando sempre as melhores soluções para cada cliente.
